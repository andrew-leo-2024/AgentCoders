### Execution Rules for Claude

1. **No aspirational language.** Every cell, every slide must contain either a number, a date, a name, or a decision. If it cannot be measured, it does not belong.
2. **Honest confidence scores.** If something is uncertain, say so with a percentage. Never present assumptions as facts.
3. **One step at a time.** Each phase has exactly ONE primary objective. Everything else is secondary.
4. **Kill criteria are mandatory.** Every phase, every sprint, every revenue stream must have explicit conditions under which it is killed.
5. **Dependencies are first-class.** Nothing floats free. Every action connects to something before it and something after it.
6. **The founder's time is the scarcest resource.** Every recommendation must account for the fact that this is a solo founder with zero capital.
7. **Slow is smooth, smooth is speed.** Methodical execution beats ambitious parallelism. Sequential proof beats simultaneous speculation.
